# Beecrowd Exercises

Repositório criado para armazenar minhas soluções dos exercícios da plataforma Beecrowd.

## Objetivo

A ideia desse repositório é:

- praticar lógica de programação
- melhorar minhas habilidades em C++
- aprender mais sobre algoritmos e estruturas básicas
- acompanhar minha evolução resolvendo problemas

## Tecnologias

- C++
- Git
- GitHub

## Estrutura

Os exercícios estão separados por arquivos e podem ser organizados futuramente por:

- nível
- categoria
- número do problema

## Plataforma

Exercícios retirados do site:

🔗 https://www.beecrowd.com.br/

## Observações

Algumas soluções podem não ser as mais otimizadas possíveis, já que o foco principal é aprendizado e prática constante.
